# online-judge-frontend

## 测试运行

`npm run dev`

## 编译

`npm run build`

## 提交

`npm run commit`

## 开发依赖变量

新建`.env.local`文件，填写：

```
VUE_APP_API_URL=http://example.com/api # devServer使用的API
```
